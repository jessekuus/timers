Hello,

This is a simple timer page made for our jiu-jitsu tournament to show the competitors when the registrations
for different open weight classes close, as it has been ongoing issue in our tournaments that competitors
(and oftentimes the organizers heh) have no idea when they have to register. 
With this, we can set the times in the morning and refer to the ongoing timers on the big screen at the venue
and be done with it.

The setup of the timer is quite rudimentary, the site takes the times from a google spreadsheet editted
from phone.

This was set up in a haste day before the tournament, so any and all changes and updates are welcome,
but as this mainly has only one function it should be easily adjustable for different tournaments
on the same federation. And usually with these, the less is more.
